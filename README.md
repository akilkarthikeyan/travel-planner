# Travel Planner

Travel Planner helps users plan trips from point A to B, with multiple stops (S1, S2, …, SN) in between, while offering flight and accommodation options at each stop. Users can apply filters for flight/Airbnb at each stop and at the end are presented with a list of plans to choose from. Plans can be sorted up/down by avg. flight/Airbnb rating, total plan price and so on.
